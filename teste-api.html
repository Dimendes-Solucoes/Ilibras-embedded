🧪 Teste de Integração API

Teste do widget com envio via POST

📡 Endpoint da API

URL: https://sistema.ilibras.com.br/administrativo/api/clientes/cadastrar.php
Método: POST
Content-Type: multipart/form-data

📤 Dados Enviados (FormData)

FormData: nome: "João Silva" cpf: "12345678900" telefone: "11987654321" token: "seu_token_aqui"

📥 Resposta Esperada

{ "status": "OK", "link_fila": "https://sistema.ilibras.com.br/acesso.php?token=...", "codigo": "ABC123", "id_atendimento": "717", "id_surdo": 417 }
⚠️ Importante

• O widget exibirá "Aguarde..." enquanto processa a requisição
• Em caso de erro, uma mensagem será exibida e o botão será reabilitado
• Após sucesso, o link_fila será aberto em nova aba
• O modal do widget fecha automaticamente e o usuário permanece na página original
• A API pode retornar link_fila, link, url ou redirect (nesta ordem de prioridade)

🎯 Como Testar

1. Clique no botão azul flutuante no canto inferior direito
2. Preencha o formulário com Nome, CPF e Telefone válidos
3. Marque a checkbox de consentimento
4. Clique em "Iniciar atendimento"
5. Aguarde o processamento
6. Uma nova aba será aberta automaticamente com o link da fila
7. O modal do widget fecha automaticamente e você permanece na página
