🧪 Teste de Integração API

Teste do widget com envio via POST

📡 Endpoint da API

URL: https://sistema.ilibras.com.br/administrativo/api/clientes/cadastrar.php
Método: POST
Content-Type: application/json

📤 Dados Enviados

{ "cpf": "12345678900", "nome_usuario": "João Silva", "nome": "João Silva" }

📥 Resposta Esperada

{ "link": "https://ilibras.com/atendimento/12345", "success": true, "message": "Cliente cadastrado com sucesso" }
⚠️ Importante

• O widget exibirá "Aguarde..." enquanto processa a requisição
• Em caso de erro, uma mensagem será exibida e o botão será reabilitado
• Após sucesso, o usuário será redirecionado automaticamente para o link retornado pela API
• A API pode retornar link, url ou redirect - todos são aceitos

🎯 Como Testar

1. Clique no botão azul flutuante no canto inferior direito
2. Preencha o formulário com Nome e CPF válido
3. Marque a checkbox de consentimento
4. Clique em "Iniciar atendimento"
5. Aguarde o processamento e redirecionamento
