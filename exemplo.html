🎯 iLibras Widget

Widget de atendimento embarcável para captura de dados

Sobre o Widget

O iLibras Widget é um componente embarcável que permite adicionar um botão flutuante (similar ao WhatsApp) em qualquer site. Quando ativado, exibe um formulário para capturar Nome, CPF e Telefone do usuário, redirecionando-o para a plataforma de atendimento com esses dados.

✨ Fácil Integração

Apenas 2 linhas de código para implementar

🎨 Personalizável

Configure cores, posição e textos

📱 Responsivo

Funciona perfeitamente em mobile e desktop

✅ Validação de Dados

Valida CPF e formata campos automaticamente

Instalação Básica

Adicione estas linhas antes do fechamento da tag </body>:

<!-- iLibras Widget --> <link rel="stylesheet" href="ilibras-widget.css"> <script src="ilibras-widget.js"></script> <script> // Inicializa o widget com configuração padrão new ILibrasWidget(); </script>
💡 Dica:

O widget aparecerá automaticamente no canto inferior direito da página.

Configuração Personalizada

Você pode personalizar o widget passando um objeto de configuração:

<script> new ILibrasWidget({ // Posição do widget position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left' // URL de redirecionamento redirectUrl: 'https://teste.com.br', // Personalização visual primaryColor: '#4A90E2', title: 'iLibras', message: 'Olá! Estamos aqui para ajudar você! 😊', buttonText: 'Iniciar atendimento', // Z-index do widget zIndex: 9999 }); </script>

Opções de Configuração

position Posição do widget na tela Opções: 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left' Padrão: 'bottom-right' redirectUrl URL para onde os dados serão enviados Padrão: 'https://teste.com.br' primaryColor Cor principal do widget (formato hexadecimal) Padrão: '#4A90E2' title Título exibido no cabeçalho do modal Padrão: 'iLibras' message Mensagem de boas-vindas Padrão: 'Olá, somos a equipe iLibras e estamos aqui para ajudar você! 😊' buttonText Texto do botão de submissão Padrão: 'Iniciar atendimento' zIndex Camada Z do widget Padrão: 9999

Como Funcionam os Dados

Quando o usuário preenche o formulário e clica em "Iniciar atendimento", ele é redirecionado para a URL configurada com os seguintes parâmetros:

// Exemplo de URL gerada: https://teste.com.br?nome=João+Silva&cpf=12345678900&telefone=11999887766&origem=meusite.com // Parâmetros enviados: - nome: Nome do usuário - cpf: CPF sem formatação (apenas números) - telefone: Telefone sem formatação (apenas números) - origem: Hostname do site de origem
🔒 Segurança:

O CPF é validado antes do envio. Números inválidos são rejeitados automaticamente.

Exemplos de Uso

Exemplo 1: Widget no canto inferior esquerdo

new ILibrasWidget({ position: 'bottom-left' });

Exemplo 2: Personalização completa

new ILibrasWidget({ position: 'bottom-right', redirectUrl: 'https://minhaplataforma.com.br/atendimento', primaryColor: '#0066CC', title: 'Suporte Online', message: 'Precisa de ajuda? Informe seus dados para iniciar o atendimento.', buttonText: 'Solicitar Atendimento' });

Exemplo 3: Configuração global (auto-inicialização)

<!-- Defina a configuração antes de carregar o script --> <script> window.iLibrasWidgetConfig = { position: 'bottom-right', redirectUrl: 'https://teste.com.br' }; </script> <script src="ilibras-widget.js"></script> <!-- Widget será inicializado automaticamente -->

Métodos Programáticos

Você pode controlar o widget via JavaScript:

// Guarde a instância do widget const widget = new ILibrasWidget(); // Abrir o widget programaticamente widget.openWidget(); // Fechar o widget programaticamente widget.closeWidget(); // Alternar (abrir/fechar) widget.toggleWidget(); // Remover o widget da página widget.destroy();
